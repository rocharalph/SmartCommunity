# CampusSafe

CampusSafe is a campus incident reporting and management system developed using Android Studio, Firebase and cloudinary.

## Features
- User Dashboard
- Incident Reporting
- Update user details
- Camera Upload
- Smart Keyword detection
- Notifications
- Admin Dashboard
- Manage Reports
- Manage Users
